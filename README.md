# Quote Parser

A simple Python script that scrapes quotes from the educational website [quotes.toscrape.com](http://quotes.toscrape.com/) and saves them to a UTF-8 encoded text file.

## Features
- Fetches quotes and their authors from the website
- Saves quotes in the format:
- '"Quote"
- Author'
- Counts the number of quotes found

## Requirements
- Python 3.x
- `requests` library
- `beautifulsoup4` library

## Installation
1. Install Python dependencies:
pip install requests beautifulsoup4

## Usage
Run the script:
python main.py
The quotes will be saved to quote.txt.
